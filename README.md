
# Caesar Cipher 

## Опис
Ця програма реалізує шифрування введеного користувачем тексту за допомогою шифру Цезаря. Створена з урахуванням вимог до структури багатофайлового проєкту на C++.

## Структура проєкту
- `functions.h` — заголовковий файл з описами функцій у namespace `Caesar`
- `functions.cpp` — реалізація функцій (введення, шифрування, виведення)
- `main.cpp` — головна функція `main()`, що викликає логіку
- `README.md` — опис проєкту

## Вимоги до запуску
- C++ компілятор (наприклад, `VS Code, CLion`)
- ОС Windows/Linux/macOS
